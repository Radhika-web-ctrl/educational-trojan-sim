# Ethical Use Agreement (MANDATORY)

Project: Educational Trojan Horse Simulation (Safe Educational Demo)

We, the undersigned, agree to the following conditions:

1) Educational Use Only  
This project is used only for academic assessment and training in a controlled environment.

2) Sandbox-Only Execution  
All simulated behaviors operate only on dummy data inside the designated sandbox directory (SANDBOX_ROOT).  
We will not run or modify this project to operate outside the sandbox.

3) No Harmful Use / No Real Malware  
We will not modify or use this project to perform real malicious actions, persistence, privilege escalation, or data exfiltration.

4) Mandatory Safeguards Must Remain Enabled  
We understand the following safeguards are mandatory and must remain intact:
- Kill switch (SANDBOX_ROOT/.kill)
- Auto-disable after 24 hours (SANDBOX_ROOT/.created_at)
- Directory whitelisting / sandbox enforcer

5) No Distribution  
This project must not be packaged, shared publicly, or distributed outside educational context.
